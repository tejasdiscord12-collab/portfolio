# Tejas Kumar Portfolio

This is a personal portfolio website showcasing my projects and skills.

## Deployment

This portfolio can be deployed to Vercel or any static hosting service.

### Vercel Deployment

1. Push this code to a GitHub repository
2. Connect the repository to Vercel
3. Vercel will automatically detect the static files and deploy them

### Manual Deployment

Simply upload all HTML, CSS, and JS files to any static hosting service.

## Project Structure

- `index.html` - Main portfolio page
- `ecommerce.html` - E-commerce project page
- `vercel.json` - Vercel routing configuration
- `package.json` - Project dependencies and scripts
